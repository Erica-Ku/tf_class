# Convolution Neural Network(CNN)
- 주로 이미지 인식 분야에서 사용되는 딥러닝 기술
- Convolution(합성곱) 계층과 Pooling(풀링) 계층으로 이루어짐
- 이미지의 공간 정보를 유지하면서 특징을 추출하는데 매우 효과적

- Convolution 계층
 => 필터 또는 커널이라는 가중치를 가지고 입력 이미지와 합성곱 연산을 수행
 => 필터는 이미지의 특징을 추출하는 역할을 수행

- Pooling 계층
 => 합성곱 계층에서 추출된 특징은 풀링 계층을 통해 축소됨
 => 이미지를 일정한 크기로 분할하고, 각 구역에서 가장 큰 값을 선택하여 새로운 이미지 생성
 => 이미지의 크기를 줄이면서도 중요한 정보를 유지할 수 있음

- 추출된 특징은 일련의 완전 연결 계층을 통해 분류 or 회귀와 같은 작업 수행 가능
 => 완전 연결 계층은 이미지의 특징을 기반으로 예측 수행
