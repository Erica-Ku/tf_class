# 텐서플로 과정 - 경사하강법 사용 => 모델의 파라미터를 업데이트하여 손실 함수를 최소화하는
				     최적의 파라미터 값을 찾는 알고리즘
 1) 데이터 불러와서 전처리
 2) 모델 아키텍처 정의 -> 파라미터의 초기값은 무작위로 설정됨
 3) 손실 함수 정의 -> 모델의 예측값과 실제값 사이의 차이를 계산함
 4) 옵티마이저 정의 -> 손실 함수를 최소화하기 위해 파라미터를 업데이트하는 방식을 결정함
 5) 모델 학습시키기 -> 학습 데이터를 모델에 입력하여 모델의 예측값을 계산하고, 
		       이를 손실 함수에 입력하여 손실 계산 -> 손실을 최소화하기 위해 옵티마이저를 사용하여 파라미터를 업데이트함
 6) 학습을 반복하면서 모델의 성능을 평가 -> 검증 데이터를 사용하여 모델의 예측값과 실제값 사이의 차이 계산
					    검증 데이터에서 손실이 줄어들지 않거나 증가하면, 학습을 중지하고 모델 평가

# 텐서플로에서 경사하강법을 사용하지 않는 경우는 극히 적음!
=> 경사하강법은 가장 기본적이면서도 핵심적인 학습 알고리즘이기 때문임
=> 경사하강법의 변형이나 개선 알고리즘을 사용할 수도 있음 -> 이것을 사용한다고 해서 경사하강법의 사용보다 좋은 결과를 보장하지는 않음

# 미니배치 경사하강법
- 경사하강법의 한 종류, 전체 데이터를 작은 묶음(미니배치)으로 나누어 처리하는 방법
- 계산량을 줄이고 더 빠르게 학습할 수 있음
- 각 미니배치를 사용하여 경사하강법을 수행하고, 모든 미니배치의 평균 기울기를 사용하여 가중치를 업데이트함
  이 과정을 모든 미니배치에 대해서 반복하며 학습을 수행함
